# プリセット一覧

## ⭐ おすすめセット

**含まれるモジュール**: Dashboards, Monitors, Logs

Datadogを初めて使う方に最適。基本的なダッシュボードとアラートが即座に使えるようになります。

## ☁️ AWS環境向け

**含まれるモジュール**: AWS, Dashboards, Monitors, APM, Logs

AWSを使っている環境向け。EC2/RDS/Lambda等のメトリクス収集、APMトレーシング、推奨アラートが一括設定されます。

## ☁️ GCP環境向け

**含まれるモジュール**: GCP, Dashboards, Monitors, APM, Logs

GCPを使っている環境向け。Compute Engine/Cloud SQL/GKE等のメトリクス収集が設定されます。Workload Identity Federation（キーレス認証）対応。

## 🔒 セキュリティ重視

**含まれるモジュール**: CSPM, CWS, ASM, SIEM, Sensitive Data Scanner

セキュリティ態勢の強化に特化。クラウドリソースのスキャン、WAFルール、脅威検出ルール、機密データスキャンを一括設定。

注意: Enterprise プラン以上が必要な機能は自動スキップされます。

## 🖥️ Xserver向け

**含まれるモジュール**: Xserver, Dashboards, Monitors

XserverのVPS/専用サーバー向け。ブラウザでXserver管理画面にログインするだけで、SSH情報の自動取得とファイアウォール設定まで完了します。

## 🚀 フル

**含まれるモジュール**: 全16モジュール

全機能を一括設定。大規模環境やフルスタック監視を構築する場合に。

## ⚙️ カスタム

Cloud / Feature / Security の3カテゴリに分けて、個別にモジュールを選択します。
